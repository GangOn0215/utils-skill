# 📋 work-log-generator

> Git 커밋 내역을 분석해서 **보고용 업무일지**를 자동으로 작성합니다.
> 일반인도 이해할 수 있는 용어로 풀어주고, [추가]/[수정]으로 자동 분류.

[← 메인 README로 돌아가기](../../README.md)

---

## 🚀 사용법

```
업무일지
일지 써줘
work log
오늘 뭐했지
커밋 정리해줘
업무 정리
```

날짜를 지정하지 않으면 **오늘 날짜**의 커밋을 자동으로 가져옵니다.

```
"5월 15일 업무일지 써줘"
"이번 주 work log"
```

---

## ✨ 주요 기능

- **토큰 효율적** — `git log --stat` 한 번으로 커밋·변경 파일을 일괄 조회, diff는 필요한 부분만
- **자동 분류** — 신규 파일/기능은 **[추가]**, 기존 코드 수정은 **[수정]**
- **사람 친화 표현** — 변수명·함수명 같은 기술 용어는 괄호로 보충 설명
- **대분류로 묶기** — 관련 커밋들을 하나의 주제(예: "파트너 프로필 관리 개편")로 그룹화

---

## 📐 출력 형식

```
## [ 프로젝트명 ] - 신규 기능
▷ 대분류 제목 (기능 단위로 묶기)
  - 소분류 상세 설명 (일반인도 이해할 수 있는 수준)
  - 소분류 상세 설명

▷ 대분류 제목
  - 소분류 상세 설명

## [ 프로젝트명 ] - 기능 수정
▷ 대분류 제목
  - 소분류 상세 설명

---

## [ 요약 ]
▷ 한두 문장으로 전체 작업 요약
```

## 💡 예시

**입력:**
> "오늘 뭐했지?"

**커밋 내역 (Claude가 자동 조회):**
- `[26.05.19] ✨ Feat: 카카오 로그인 연동`
- `[26.05.19] 🐛 Fix: 파트너 프로필 사진 업로드 오류 수정`
- `[26.05.19] ♻️ Refactor: 시공 후기 작성 화면 컴포넌트 분리`

**출력:**

```
## [ 인테리어 플랫폼 ] - 신규 기능
▷ 소셜 로그인 확장
  - 카카오톡 계정으로 로그인할 수 있는 기능 추가
  - 신규 가입 시 카카오 프로필 정보로 기본값 자동 설정

## [ 인테리어 플랫폼 ] - 기능 수정
▷ 파트너 프로필 관리 안정화
  - 파트너 회원의 프로필 사진 업로드가 실패하던 문제 해결

▷ 시공 후기 작성 화면 개편
  - 작성 화면 내부 구조를 재정비하여 향후 수정·확장이 쉬워짐
  - 사용자가 보는 화면 자체는 동일

---

## [ 요약 ]
▷ 카카오 소셜 로그인을 새로 도입하고, 파트너 프로필 사진 업로드 버그를 잡았으며,
  시공 후기 작성 화면 내부 구조를 정비했습니다.
```

---

## 📏 작성 규칙

- **대분류(▷)**: 관련 기능을 한 주제로 묶기 (예: "결제 모듈 안정화")
- **소분류(-)**: 일반인이 이해할 수 있는 풀어쓰기, 2칸 들여쓰기
- 추가/수정 중 해당사항 없는 섹션은 생략
- 단순 주석·공백 변경은 통합하거나 생략
- 마지막에 `## [ 요약 ]`으로 한두 문장 요약
